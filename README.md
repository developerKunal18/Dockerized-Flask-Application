# Dockerized Flask Application

A Flask application packaged using Docker.

## Features

- Docker containerization
- Portable deployment
- Dependency isolation
- Environment consistency

## Technologies Used

- Python
- Flask
- Docker

## Installation

Build image:

```bash
docker build -t flask-app .
```

Run container:

```bash
docker run -p 5000:5000 flask-app
```

Open:

```text
http://localhost:5000
```

## Purpose

focuses on containerization and deployment fundamentals.
